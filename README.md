# Feria_vlc
Todos los supuestos de las practicas de feria valencia

# Supuesto 1
Creación de un boletín informativo para Salón del Cómic de València, destacando noticias y promociones especiales.

---

# Objetivo
Mi objetivo es redactar un correo que no solo transmita información clara y útil sobre el evento, sino que también logre captar la atención e interés de aquellas personas que aún no conocen el Salón del Cómic de València, despertando su curiosidad y motivándolas a descubrir todo lo que este evento tiene para ofrecer.
